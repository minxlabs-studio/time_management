# `styles/` files

This folder is the target home for CSS extracted from the `<style>` block in `index.html`.

## Suggested split

- `base.css`: reset, variables, body, typography.
- `layout.css`: app shell, topbar, tabs, content, modal, toast, utility buttons.
- `tasks.css`: weekly matrix, task list, add task forms, stats.
- `calendar.css`: Google Calendar page, week strip, calendar selector, event form.
- `goals.css`: goals page, goal cards, milestones, progress bars.
- `ai-coach.css`: AI Coach buttons, input, response, spinner.
- `memories.css`: memory jar UI.
- `review.css`: weekly review and hour calculator.

## Migration rule

Move one CSS section at a time and test after every move.
